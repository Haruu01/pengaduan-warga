Langkah Instalasi

Ikuti langkah berikut untuk menginstall aplikasi

1
Download Source Code

Download file ZIP atau clone repository dari GitHub

# Via Git
git clone https://github.com/Haruu01/pengaduan-warga.git

# Atau download ZIP dari GitHub
2
Setup Database

Buat database MySQL dan import struktur tabel

CREATE DATABASE pengaduan_warga;
mysql -u username -p pengaduan_warga < database/pengaduan_warga.sql
3
Konfigurasi Database

Edit file konfigurasi database di app/config/config.php

define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'pengaduan_warga');
define('BASE_URL', 'http://yourdomain.com/');
4
Upload ke Hosting

Upload semua file ke hosting dan set permissions

# Set permissions untuk upload directory
chmod 755 public/
chmod 777 public/uploads/
5
Test Aplikasi

Akses aplikasi dan test login admin

System Requirements

Pastikan hosting Anda memenuhi persyaratan berikut

Server Requirements
  • PHP 7.4 atau lebih tinggi
  • MySQL 5.7 atau lebih tinggi
  • Apache/Nginx web server
  • mod_rewrite enabled
PHP Extensions
  • PDO MySQL
  • GD Library
  • mbstring
  • fileinfo
